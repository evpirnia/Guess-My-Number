# Guess-My-Number

A Guess-My-Number implementation in Lisp. The user picks a number from 1 to 100 and aids the computer in guessing the user's number by entering (smaller) or (bigger).

## Run
```bash
$ clisp
> load(epirnia1.lisp);
> (guess-my-number)
>
```

## Start a New Game
```bash
> (start-over)
```

## Exit
```bash
> (exit)
```
